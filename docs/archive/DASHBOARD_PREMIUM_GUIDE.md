## ğŸš¨ Troubleshooting

### Problem: Dashboard not opening
**Solution:**
```bash
# Check if port 5000 is free
lsof -i :5000

# If busy, kill the process or use different port
python dashboard_premium.py
```

### Problem: No leads showing
**Solution:**
1. Generate leads first using "Generate Premium Leads" button
2. Wait for generation to complete (10-15 minutes)
3. Refresh page

### Problem: WhatsApp not opening
**Solution:**
- Make sure WhatsApp Web is logged in
- Check if phone number is valid
- Try copying message manually

### Problem: Email not opening
**Solution:**
- Make sure you have default email client set
- Try copying content manually
- Use Gmail/Outlook web interface

### Problem: Generation taking too long
**Solution:**
- This is normal! 50 leads = 10-15 minutes
- Check progress bar for status
- Don't close browser during generation
